2.1 题目1：图的遍历
2.1.1 问题描述
给定一个无向图，用 DFS 和 BFS 找出图的所有连通分量，顶点编号 0..n-1，搜索从编号最小的顶点开始。输出格式示例：{0 1 12}{3 4}...

2.1.2 基本要求
- 使用邻接矩阵或邻接表（邻接表要求尾插法）
- DFS 与 BFS 的遍历顺序需从当前未访问的最小编号顶点开始

2.1.3 数据结构设计
- 使用邻接矩阵表示图：vector<vector<int>>，便于按编号顺序遍历邻居
- visited 标记访问状态
- 使用 stack 模拟递归的 DFS，queue 实现 BFS

2.1.4 功能说明（函数说明）
- `dfs_component(...)`：从起点出发，使用栈 DFS 遍历一个连通分量并记录顶点序列
- `bfs_component(...)`：从起点出发，使用队列 BFS 遍历一个连通分量并记录顶点序列
- `print_components(...)`：按要求格式输出所有连通分量

2.1.5 关键代码实现（摘自项目文件 [1.cpp](1.cpp)）
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <stack>
using namespace std;

void bfs_component(const vector<vector<int>> &graph, int start, vector<bool> &visited, vector<int> &component)
{
    queue<int> q;
    q.push(start);
    visited[start] = true;

    while (!q.empty())
    {
        int node = q.front();
        q.pop();
        component.push_back(node);

        for (int i = 0; i < graph.size(); i++)
        {
            if (graph[node][i] == 1 && !visited[i])
            {
                visited[i] = true;
                q.push(i);
            }
        }
    }
}

void dfs_component(const vector<vector<int>> &graph, int start, vector<bool> &visited, vector<int> &component)
{
    stack<int> s;
    s.push(start);

    while (!s.empty())
    {
        int node = s.top();

        if (!visited[node])
        {
            visited[node] = true;
            component.push_back(node);
        }

        int next_unvisited = -1;
        for (int i = 0; i < graph.size(); i++)
        {
            if (graph[node][i] == 1 && !visited[i])
            {
                next_unvisited = i;
                break;
            }
        }

        if (next_unvisited != -1)
            s.push(next_unvisited);
        else
            s.pop();
    }
}

// main() 中构建邻接矩阵并分别调用 DFS/BFS 计算所有连通分量，最后输出
```

2.1.6 调试分析
- 在邻接矩阵遍历邻居时按编号从小到大检查，保证输出的连通分量内部顺序符合题目要求。遇到重边不影响结果。

2.1.7 总结和体会
- 使用邻接矩阵便于按编号顺序遍历邻居，但空间复杂度是 O(n^2)。对 n<=1000 的数据在本题范围内可接受。

2.2 题目2：六度空间
2.2.1 问题描述
对社交网络图的每个结点，计算与该结点距离不超过 6 的结点占总数的百分比，按 float 计算并保留两位小数。

2.2.2 基本要求
- 输入节点编号从 1..N，转换为 0..N-1 存储
- 对每个节点使用 BFS 到深度 6

2.2.3 数据结构设计
- 邻接表存储图：vector<vector<int>>，节省空间并高效遍历实际邻居

2.2.4 关键代码实现（摘自项目文件 [2.cpp](2.cpp)）
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <iomanip>
using namespace std;

int count_within_6(int start, const vector<vector<int>> &adj_list)
{
    int n = adj_list.size();
    vector<bool> visited(n, false);
    queue<pair<int, int>> q; // node, distance

    q.push({start, 0});
    visited[start] = true;
    int count = 1;

    while (!q.empty())
    {
        int node = q.front().first;
        int distance = q.front().second;
        q.pop();
        if (distance >= 6) continue;
        for (int neighbor : adj_list[node])
        {
            if (!visited[neighbor])
            {
                visited[neighbor] = true;
                q.push({neighbor, distance + 1});
                count++;
            }
        }
    }

    return count;
}

// main() 中按题目读取图，循环调用 count_within_6 并输出百分比
```

2.2.5 总结
- 每次 BFS 限制深度 6，复杂度为 O(N + M)（在实际邻居数基础上），对 N<=2000 可接受。

2.3 题目3：村村通（最小生成树加已有边）
2.3.1 问题描述
给出村庄间距离矩阵，部分边已经建成（视为 0 成本），再新建若干道路使得所有村庄连通且新建长度之和最小。

2.3.2 基本要求
- 将已存在的边视为权重 0，求最小生成树（MST）总权值。

2.3.3 关键代码实现（摘自项目文件 [3.cpp](3.cpp)）
```cpp
#include <iostream>
#include <vector>
#include <climits>
using namespace std;

int prim(int n, vector<vector<int>> &graph, const vector<pair<int, int>> &existing_edges)
{
    vector<int> dist(n, INT_MAX);
    vector<bool> visited(n, false);

    for (const auto &edge : existing_edges)
    {
        int u = edge.first;
        int v = edge.second;
        graph[u][v] = 0;
        graph[v][u] = 0;
    }

    dist[0] = 0;
    int total_cost = 0;

    for (int i = 0; i < n; i++)
    {
        int min_dist = INT_MAX;
        int u = -1;
        for (int j = 0; j < n; j++)
            if (!visited[j] && dist[j] < min_dist) { min_dist = dist[j]; u = j; }

        visited[u] = true;
        total_cost += dist[u];

        for (int v = 0; v < n; v++)
            if (!visited[v] && graph[u][v] < dist[v]) dist[v] = graph[u][v];
    }

    return total_cost;
}

// main() 中读取距离矩阵、已存在边，调用 prim()
```

2.3.4 总结
- 使用 Prim 算法并将已有边权重设为 0，时间复杂度 O(n^2)，适用于 n<=100 的约束。

2.5 题目5：小马吃草
2.5.1 问题描述
每匹小马需先到任意有牧草点，再到目标点，求最短路程。图连通且至少有一个牧草点。

2.5.2 基本思路
- 使用 Dijkstra 求单源最短路径
- 采用多源 Dijkstra 预处理每个点到最近牧草点的距离（也可对每组查询运行 Dijkstra）

2.5.3 关键代码实现（摘自项目文件 [5.cpp](5.cpp)）
```cpp
#include <iostream>
#include <vector>
#include <queue>
#include <climits>
using namespace std;

typedef long long ll;
const ll INF = 1e18;

vector<ll> dijkstra(int start, const vector<vector<pair<int, int>>> &graph, int n)
{
    vector<ll> dist(n + 1, INF);
    priority_queue<pair<ll, int>, vector<pair<ll, int>>, greater<pair<ll, int>>> pq;
    dist[start] = 0;
    pq.push({0, start});
    while (!pq.empty())
    {
        ll d = pq.top().first;
        int u = pq.top().second;
        pq.pop();
        if (d > dist[u]) continue;
        for (auto &edge : graph[u])
        {
            int v = edge.first, w = edge.second;
            if (dist[u] + w < dist[v]) { dist[v] = dist[u] + w; pq.push({dist[v], v}); }
        }
    }
    return dist;
}

// main() 中构建无向带权图，先用多源 Dijkstra 计算每点到最近牧草点的距离
// 每个查询可用两次 Dijkstra（start/end）或枚举牧草点计算 start->grass + grass->end
```

2.5.4 总结
- 题中 N<=1000，M<=2N，Dijkstra 多次运行在时间上可接受；多源 Dijkstra 用于计算到最近牧草点的距离，节约重复计算。
